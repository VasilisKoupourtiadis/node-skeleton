1. **Klona repot:**
   - Öppna terminalen.
   - Navigera till mappen för Webbserver-kursen och kör följande kommando:

```bash
git clone https://github.com/VasilisKoupourtiadis/node-skeleton.git
```

2. **Installera dependencies:**

```bash
cd .\node-skeleton\
```

```bash
code .
```

```bash
npm i
```

3. **Starta projektet**

```bash
npm run dev
```
